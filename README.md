# Econ 30

This repository contains materials for the Econ 30 lab sections at Stanford (Spring 2026).

## Contents
- Section slides  
- Coding tutorials  
- Examples and exercises  
- Resources for building a knowledge base  

## Tools
We will use:
- GitHub for version control  
- Cursor / VS Code for coding  
- Python for data work  

## Purpose
This repo is meant to help students build practical skills in:
- organizing code and projects  
- using AI tools effectively  
- working with real data
